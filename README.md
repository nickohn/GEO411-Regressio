# GEO411-Regression
R analysis of regression

R was used to conduct linear regression analysis on data on the number of moquito pupae and development sites available as well as on a separate dataset of SAT scores predicting GPA.

A double log analysis of the mosquito dataset led to a failure to reject the null hypothesis with a p-value of <2.2e-16

A regression analysis of SAT scores and GPA led to a failure to reject the null hypothesis with a p-value of 0.000152

These tests were conducted using R Studio on MacOS.
